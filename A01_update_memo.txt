Understanding Version Numbering
version [Main. Sub. Last]
-- Main: Indicates changes in the Core Module [pivdataprocessor]. An immediate update is strongly preferable.  
-- Sub: Indicates changes in Examples. Update is recommended.
-- Last: Minor changes or daily supplyment. Updating is optional.

Note that: 
All updates are forward-compatible and based on the immediately preceding version.
